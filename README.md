# IMPRESIÓN SUBLIME PTY

Sitio web oficial del emprendimiento IMPRESIÓN SUBLIME PTY.

## 🚀 Descripción

Esta página web fue creada para mostrar y promocionar productos personalizados mediante sublimación, incluyendo:

- Camisas personalizadas
- Tazas sublimadas
- Gorras
- Vinil textil
- Diseños personalizados
- Regalos personalizados

---

## 🌐 Objetivo

Ofrecer una tienda virtual sencilla, rápida y accesible para que los clientes puedan visualizar productos y realizar pedidos fácilmente por WhatsApp.

---

## 🛠️ Tecnologías utilizadas

- HTML5
- CSS3
- JavaScript
- GitHub Pages

---

## 📱 Características

✅ Diseño responsive  
✅ Compatible con celulares  
✅ Integración con WhatsApp  
✅ Catálogo de productos  
✅ Fácil de editar  
✅ Hosting gratuito con GitHub Pages
